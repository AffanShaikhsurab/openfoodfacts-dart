Returns the image full url - for a specific imageSize if needed.

Returns null is barcode is null. E.g. "https://static.openfoodfacts.org/images/products/359/671/046/2858/front_fr.4.100.jpg"

Implementation

static String? buildUrl(
  final String? barcode,
  final ProductImage image, {
  final ImageSize? imageSize,
  final QueryType? queryType,
  final String? root,
}) =>
    barcode == null
        ? null
        : '${getProductImageRootUrl(
            barcode,
            queryType: queryType,
            root: root,
          )}'
            '/'
            '${getProductImageFilename(
            image,
            imageSize: imageSize,
          )}';
